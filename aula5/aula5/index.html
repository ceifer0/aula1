HTML Linguagem de Marcação de texto

tutoriais para iniciantes

HTML (Linguagem de Marcação de Hipertexto) é o bloco de construção mais básico da WEB. define o
significado e a estrutura do contúdo web. Outras tecnologias além do HTML geralmente são
usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página web.
